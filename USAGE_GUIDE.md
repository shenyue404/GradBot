# 毕业设计辅助助手 - 使用指南

## 项目概述

这是一个前后端分离的毕业设计管理系统，帮助教师管理学生毕业设计的全流程，包括学生信息管理、选题指导、任务书生成、开题报告指导、中期报告检查和评审答辩支持等功能。

## 访问地址

### 前端界面
- **开发环境**: http://localhost:5173
- **生产环境**: 根据部署配置而定

### 后端API
- **开发环境**: http://localhost:3000
- **API文档**: http://localhost:3000/api/health (健康检查)

## 快速开始

> ⚠️ **重要**：首次使用前，请先阅读 `SETUP_GUIDE.md` 完成所有配置！

### 1. 环境准备

确保已安装：
- Node.js >= 16.0.0
- MySQL >= 5.7 或 MySQL 8.0+
- npm 或 yarn

### 2. 数据库初始化

```bash
# 登录MySQL并创建数据库
mysql -u root -p < backend/database/schema.sql

# （可选）导入示例数据
mysql -u root -p gradbot < backend/database/init.sql
```

### 3. 配置环境变量

在 `backend` 目录下创建 `.env` 文件：

```env
PORT=3000

# MySQL 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_NAME=gradbot
DB_USER=root
DB_PASSWORD=your_password_here

# OpenAI API 配置（用于AI功能，可选）
OPENAI_API_KEY=your_openai_api_key_here

NODE_ENV=development
```

**详细配置说明请参考 `SETUP_GUIDE.md`**

### 4. 安装依赖

```bash
# 后端
cd backend
npm install

# 前端（新开一个终端）
cd frontend
npm install
```

### 5. 启动服务

**启动后端**（在 `backend` 目录）：
```bash
npm run dev
```
看到 "MySQL 数据库连接成功" 和 "服务器运行在端口 3000" 表示启动成功

**启动前端**（在 `frontend` 目录，新开一个终端）：
```bash
npm run dev
```
看到 "Local: http://localhost:5173" 表示启动成功

### 6. 访问系统

打开浏览器访问：**http://localhost:5173**

**如果遇到问题，请查看 `SETUP_GUIDE.md` 中的常见问题解决方案**

## 功能模块使用说明

### 1. 学生信息管理

**访问路径**: 左侧菜单 → "学生信息管理"

**功能**:
- ✅ 单个录入：点击"新增学生"按钮，填写学生信息
- ✅ 批量导入：点击"批量导入"，上传Excel文件（格式：学号、姓名、专业、班级、邮箱、电话）
- ✅ 搜索：支持按学号、姓名搜索，按专业筛选
- ✅ 编辑：点击表格中的"编辑"按钮
- ✅ 删除：点击表格中的"删除"按钮

**Excel导入格式示例**:
| 学号 | 姓名 | 专业 | 班级 | 邮箱 | 电话 |
|------|------|------|------|------|------|
| S001 | 张三 | 计算机科学 | 计科2021-1 | zhang@example.com | 13800138000 |

### 2. 选题指导

**访问路径**: 左侧菜单 → "选题指导"

**功能**:
- ✅ 智能生成选题：
  1. 点击"智能生成选题"按钮
  2. 输入研究方向（如：人工智能、机器学习）
  3. 输入关键词（用逗号、顿号或空格分隔）
  4. 点击"生成选题"
  5. 从生成的选题中选择一个，点击"选择此选题"
- ✅ 手动创建：点击"手动创建"按钮
- ✅ 关联学生：点击"关联学生"按钮，将选题与学生关联
- ✅ 导出Excel：点击"导出Excel"按钮

### 3. 任务书生成

**访问路径**: 左侧菜单 → "任务书生成"

**功能**:
- ✅ 生成任务书：
  1. 点击"生成任务书"按钮
  2. 选择学生
  3. 选择该学生的选题
  4. 系统自动生成任务书初稿
- ✅ 编辑任务书：点击表格中的"编辑"按钮
- ✅ 导出Word：点击"导出Word"按钮，下载任务书文档

### 4. 开题报告指导

**访问路径**: 左侧菜单 → "开题报告指导"

**功能**:
- ✅ 上传开题报告：点击"上传开题报告"按钮，上传文件
- ✅ 查看详情：点击"查看详情"查看分析结果
- ✅ 生成参考意见：点击"生成参考意见"按钮

**注意**: 上传文件后，系统会自动分析报告内容，识别问题并提供修改建议。

### 5. 中期报告检查

**访问路径**: 左侧菜单 → "中期报告检查"

**功能**:
- ✅ 新增中期报告：点击"新增中期报告"按钮
- ✅ 填写进度和关键点评
- ✅ 生成报告：点击"生成报告"按钮，系统自动生成分析报告
- ✅ 查看详情：点击"查看详情"查看完整报告

### 6. 评审与答辩支持

**访问路径**: 左侧菜单 → "评审与答辩支持"

**功能**:
- ✅ 论文评审：
  1. 点击"论文评审"按钮
  2. 选择学生
  3. 选择评审者类型（自己学生/其他学生）
  4. 填写总体情况和关键点评
  5. 系统自动生成评审参考意见
- ✅ 答辩评审：
  1. 点击"答辩评审"按钮
  2. 选择学生
  3. 填写总体情况和关键点评
  4. 系统自动生成答辩参考意见
- ✅ 查看详情：点击"查看详情"查看完整的评审意见

## 常见问题

### 1. 后端启动失败

**问题**: `MySQL 数据库连接失败`

**解决方案**:
- 检查 MySQL 服务是否启动
- 检查 `.env` 文件中的数据库配置是否正确
- 确认数据库 `gradbot` 已创建

### 2. 前端无法连接后端

**问题**: 前端页面显示网络错误

**解决方案**:
- 确认后端服务已启动（http://localhost:3000）
- 检查 `frontend/vite.config.ts` 中的代理配置
- 查看浏览器控制台的错误信息

### 3. AI功能无法使用

**问题**: 选题生成、任务书生成等功能报错

**解决方案**:
- 检查 `.env` 文件中是否配置了 `OPENAI_API_KEY`
- 确认 API Key 有效且有足够的额度
- 查看后端控制台的错误日志

### 4. 文件上传失败

**问题**: Excel导入或文件上传失败

**解决方案**:
- 检查 `backend/uploads/` 目录是否存在且有写权限
- 确认文件格式正确（Excel: .xlsx, .xls）
- 检查文件大小是否超过限制（10MB）
